Deskripsi Aplikasi

Aplikasi web ini merupakan aplikasi manajemen transaksi sederhana yang dirancang untuk membantu pengguna dalam mencatat, mengelola, dan melihat data transaksi secara terstruktur melalui antarmuka berbasis web. Aplikasi dibangun menggunakan HTML sebagai struktur halaman, CSS untuk pengaturan tampilan dan responsivitas, serta JavaScript untuk validasi formulir dan interaksi pengguna.

Aplikasi memiliki beberapa halaman utama, yaitu Beranda, Transaksi, Daftar Transaksi, Profil, dan Login. Navigasi antar halaman disediakan melalui menu navigasi pada bagian header sehingga memudahkan pengguna dalam mengakses setiap fitur aplikasi. Header juga dilengkapi dengan logo aplikasi sebagai identitas visual.

Halaman Beranda berfungsi sebagai halaman utama yang menampilkan informasi umum serta panduan singkat mengenai langkah-langkah penggunaan aplikasi. Halaman Transaksi menyediakan formulir pengisian data transaksi yang dilengkapi dengan validasi menggunakan JavaScript untuk memastikan seluruh input telah diisi dengan benar sebelum disimpan. Data transaksi yang dimasukkan dapat ditampilkan dalam bentuk daftar pada halaman Daftar Transaksi, dengan informasi seperti nama transaksi, jumlah, dan kategori.

Halaman Profil berisi deskripsi singkat mengenai aplikasi yang dibuat, termasuk tujuan dan manfaat aplikasi. Sementara itu, halaman Login digunakan sebagai simulasi autentikasi pengguna sebelum mengakses aplikasi.

Aplikasi ini dirancang dengan konsep responsive design menggunakan media query, sehingga tampilan dapat menyesuaikan dengan berbagai ukuran layar, baik pada perangkat desktop maupun mobile. Secara keseluruhan, aplikasi ini bertujuan untuk menerapkan konsep dasar pengembangan web front-end yang terstruktur, interaktif, dan user-friendly sesuai dengan ketentuan tugas.
